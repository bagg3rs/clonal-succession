Time Display Test

This test verifies that the time and day display feature works correctly in the simulation.

The simulation should show the current simulation time and day in the format "Day X, HH:MM".

The time should advance at a rate determined by the simulation speed.

Expected Behavior:

  • The simulation time should start at "Day 0, 00:00"
  • The time should advance as the simulation runs
  • Changing the simulation speed should affect the rate at which time advances
  • When the simulation speed is greater than 1, the speed should be displayed next to the time
  • Resetting the simulation should reset the time to "Day 0, 00:00"
